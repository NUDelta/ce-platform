## Style Guide

Because we're cool and want to be ES6 friendly.

### Variable Declarations
Use `let` instead of `var`.

```js
// bad
var foo = 'bar';

// good
let foo = 'bar';
```

### Quotations
Use single quotes for Javascript, and double quotes for HTML.

```js
// bad
let foo = "bar";

// good
let foo = 'bar'
```
